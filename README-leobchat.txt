leobChat - Chatbot do Artista Leo Barbosa
====================================

O que é?
--------
O leobChat é um chatbot inteligente para o site do Professor Leo (leob.) que funciona como um assistente virtual para tirar dúvidas dos alunos e visitantes sobre arte, pintura hiper-realista e as obras do catálogo.

Como funciona?
--------------
O leobChat responde com base em materiais reais. Ele tem um "cérebro" (vector store) alimentado com PDFs que contêm:

- Catálogo de obras Fine Art
- Informações sobre o curso de efeito metálico
- Biografia e trajetória do Professor Leo
- Certificado com especificações técnicas das impressões
- Perguntas estratégicas sobre dores dos artistas

Formas de uso?
--------------
O leobChat pode ser usado de duas formas:

1. Chat standalone em leobchat.vercel.app
2. Widget flutuante (botão laranja) que abre no canto do site do Artista Leo quando alguém clica

Tecnologias?
------------
- API Routes na Vercel
- OpenAI Responses API
- Vector Store para busca semântica
- Frontend com Tailwind CSS

Objetivo?
---------
O leobChat ajuda a converter visitantes em alunos e compradores de arte, respondendo dúvidas sobre as obras, o curso e o leob de forma natural e conversacional.
